# قالب PDF التسريبات — مدارك النخبة

## الغرض
بناء PDF احترافي لتسريبات اختبار التحصيلي (وقابل لإعادة الاستخدام للفترات القادمة: الخميس، الجمعة، ...).

## الملفات
- `build_tahsili_pdf.py` — السكربت الرئيسي
- `fonts/` — IBM Plex Sans Arabic (Bold/SemiBold/Medium/Regular)

## كيفية الاستخدام للفترات القادمة

### الخطوة ١: حدّث leak_group_id
في `build_tahsili_pdf.py` ابحث عن:
```python
url = f"{SB}/rest/v1/questions?leak_group_id=eq.cfaf82ac-dc99-43ac-8d00-d44133802245..."
```
استبدل `cfaf82ac-...` بـUUID الـleak_group الجديد.

### الخطوة ٢: حدّث العنوان والتاريخ في `draw_cover`
- "اليوم الأول" → "اليوم الثاني" / "اليوم الثالث"
- "الأربعاء ١٣ مايو" → التاريخ الجديد

### الخطوة ٣: حدّث العداد الإجمالي
- "188" → عدد الأسئلة الفعلي (الـclean بعد فلترة figure-dependent)
- chips: [الرياضيات/الفيزياء/الكيمياء/الأحياء] → الأعداد الفعلية

### الخطوة ٤: شغّل
```bash
cd ~/Projects/madarek-pdf  # أو المسار المحلي
python3 build_tahsili_pdf.py
```

## مواصفات الـPDF
- **المقاس**: A5 portrait (420×595 pt) — مناسب للجوال
- **الخط**: IBM Plex Sans Arabic (Regular/Medium/SemiBold/Bold)
- **الألوان** (مطابقة الموقع):
  - بنفسجي عميق: `#3326A0 → #4B3EC4 → #5B4DD4` (gradient الغلاف)
  - بنفسجي البراند: `#6D5DF6` (primary)
  - بنفسجي متوسط: `#5B4DD4` (deep)
  - ذهبي عميق: `#D97706` (للأرقام والـCTA)
  - ذهبي ساطع: `#F59E0B` (للأكسنت)
  - أصفر شرح: `#FEFCE8` (خلفية) + `#FDE68A` (border)
  - أخضر صحيح: `#22C55E` + `#DCFCE7` (خلفية)
- **الشعار**: ابيض مرسوم بـSVG paths (لا PNG بمربع بنفسجي)

## المبادئ التصميمية
1. **typography-first** — لا عناصر زخرفية عشوائية (دوائر، blobs)
2. **breathing space** — مسافة كافية بين كل block
3. **2 خطوط ذهبية رفيعة** تفصل info-block (188 + chips)
4. **smart pagination** — السؤال لا يُقطع بين صفحتين
5. **أرقام لاتينية** (1, 2, 188) — أوضح وأشمل
6. **شعار أبيض في الهيدر** — مندمج مع البنفسجي
7. **QR لكل سؤال** يحوّل لـ`madarekelite.com/q/{id}`

## فلترة الأسئلة (مهم)
قبل البناء، استبعد الأسئلة اللي تحتوي على:
- "في الشكل" / "الشكل المجاور" / "الشكل التالي"
- "الرسم البياني المجاور"
- "التمثيل البياني المجاور"
- "الأشكال التالية"
- "الجدول التالي"

(ما لم يكن لها `image_url`).

## بنية الـPDF
1. **الغلاف** — بنفسجي gradient + شعار + tagline + إصدار حصري + العنوان + اليوم/التاريخ + 188 + chips + CTA + tagline سفلي
2. **فاصل لكل مادة** — صفحة كاملة بنفسجي (الرياضيات/الفيزياء/الكيمياء/الأحياء)
3. **صفحات الأسئلة** (~22 صفحة لكل مادة عادةً) — header بنفسجي + ٢-٣ أسئلة/صفحة + footer
4. **الختام** — شكر + ميزات + CTA + واتساب الدعم
